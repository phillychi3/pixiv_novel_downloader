# Pixiv bookmark downloader

## install

```bash
pip install -r ".\requirements.txt"

```
## use 
first:
```
python main.py
```

1. open https://www.pixiv.net/
2. open devtools
3. switch to network page
4. copy cookie to novel.json
5. edit novel.json
6. find youe user id
